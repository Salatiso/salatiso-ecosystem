Frequently Asked Questions

How do I update my profile?

Navigate to the Profile section from the sidebar, fill in your information, upload pictures, and click save.

What is the maximum file size for pictures?

The maximum file size for profile pictures is 10MB each, and you can upload up to 5 pictures.

How do I reset my password?

Go to Settings > Security and click "Reset Password". You will receive an email with instructions.

Can I export my profile?

Yes! Go to Profile > File Management and click "Export Profile" to download a JSON backup.

Is my data secure?

Yes, we use HTTPS encryption and follow industry-standard security practices. See Privacy Settings for more.
