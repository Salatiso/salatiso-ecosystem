Webhook Configuration

Set up real-time notifications for safety events and system updates.

Available Webhook Events

  • emergency.activated - Emergency response triggered
  • safety.incident - Safety incident reported
  • trust.verified - Trust verification completed
  • user.joined - New user registration
  • system.alert - System maintenance notifications

Webhook Payload Example

{
"event": "emergency.activated",
"timestamp": "2024-09-17T10:30:00Z",
"data": {
"userId": "user_123",
"location": {"lat": -26.2041, "lng": 28.0473},
"responseTime": "2.5s"
}
}
